# InternAI 🎯
AI-powered internship intelligence platform built for Anakin Build-a-thon.

## Live Demo
https://intern-ai-one.vercel.app

## What it does
Search internships across RemoteOK, WeWorkRemotely & Indeed 
with AI powered salary insights and career recommendations.

## Built with
- Anakin Wire APIs (RemoteOK, WeWorkRemotely, Indeed)
- Google Gemini AI
- React + Vite + Tailwind
- FastAPI + Python

## Architecture
User → React Frontend → FastAPI Backend → Anakin Wire → AI Analysis → Dashboard
